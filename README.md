## 🍽️ MealDetails Modal Component
A responsive and elegant React modal component that displays detailed information about a meal fetched from <b>TheMealDB API </b>.

## ✨ Features
<ul>
<li>Fetches complete meal data using mealId
<li>Displays high-quality meal image
<li>Dynamically generates a clean ingredients list with measurements
<li>Shows full cooking instructions with preserved line breaks
<li>Includes direct link to YouTube tutorial (when available)
<li>Fully responsive modal built with Bootstrap 5
<li>Clean loading state and error handling
<li>Accessible close button and keyboard-friendly </li>
</ul>

## 🛠️ Technologies Used
<ul>
<li>React (Hooks: useState, useEffect)
<li>Axios (for API calls)
<li>Bootstrap 5 (for styling and modal layout)
<li>TheMealDB API </li> </ul>

## 🔧 Installation & Setup
1- Make sure you have the required dependencies: </br>
npm install axios bootstrap

2- Then import Bootstrap CSS in your index.js or App.js: </br>
import "bootstrap/dist/css/bootstrap.min.css";
